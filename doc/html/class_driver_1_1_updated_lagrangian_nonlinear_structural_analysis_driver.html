Detailed Description

This class derives from the NonlinearTransientAnalysisDriver class and (re)implements some methods to suit well for use for a nonlinear structural analysis. Nonlinear structural analysis uses a time parameter to define the load steps for a static solution. The transient analysis with inertial forces is also dependent on the same approach.
