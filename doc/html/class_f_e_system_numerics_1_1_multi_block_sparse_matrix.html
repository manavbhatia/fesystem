Detailed Description

template<typename T>
class FESystemNumerics::MultiBlockSparseMatrix< T >

This class allows construction of a sparse matrix using multiple sparse matrix blocks. The sparse matrix blocks should be initialized and given to this matrix. The sparsity pattern of each one of these matrices will be used to create the sparsity pattern of this matrix
