Detailed Description

This class will store all the loads and will provide methods to obtain loads based on certain criterion, which could be load case, time instant, element ID, load tag, etc. The results for all these will be returned as a vector of Load. The idea of this class is similar to that of a database where you store all the data, perform a search on it, and return the results.
