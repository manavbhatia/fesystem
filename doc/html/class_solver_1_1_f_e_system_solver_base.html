Detailed Description

this class will provide an interface to various solution algorithms available in the solver package. It can also be used to create an algorithm that is unavailable in the solver package, and can still have a uniform interface through the class methods each solver object will need a set of matrices and vectors for the solution process, which will vary depending upon the solver type. Before each solver step, pointers to these matrices and vectors will have to be set.
