Detailed Description

This class provides methods to calculate the matrices and vectors related to the thermal discipline. this is a problem with this class. It adds a variable to the mesh/dofmap, in addition to the ones already added by the structural/thermal disciplines, hence distorts the dof numbering, etc. Hence, these variables should be added to an independent dofmap, without messing with the thermal/structural mesh. It basically needs a separate dofmap structure.
