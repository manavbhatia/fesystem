Detailed Description

template<typename EnumClass>
struct FESystemElem::FESystemElemTypeEnum::EnumRegistration< EnumClass >

this defines an object that will serve the purpose of registering the object to the object handler. It is important that a static object of this class be added inside every derived enum class, since it will register the enum class with this base.
